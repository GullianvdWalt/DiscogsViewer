# DiscogsViewer
A Spring Boot + VueJs demo application for viewing content from Discogs API

## Technologies Used
* Spring Boot 2.4.5
* Vue.js 3
* Vuex
* Vuetify
* MySQL 

## Dependencies 
* Spring Secuirty
* Spring Web
* MySQL Connector
* Spring Boot JPA
* Spring Boot OAuth
* Thymeleaf

## API
[Discogs API](https://www.discogs.com/developers)
